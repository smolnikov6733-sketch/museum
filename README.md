# Museum Digital

Интерактивный сайт-агрегатор музеев мира (курсовая работа по веб-разработке).

## Публичный сайт

После включения GitHub Pages сайт доступен по адресу:

**https://smolnikov6733-sketch.github.io/museum/**

## Локальный запуск

Откройте `index.html` в браузере или запустите простой сервер:

```bash
python3 -m http.server 8080
```

Затем откройте http://localhost:8080
